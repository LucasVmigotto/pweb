1. Descreva as motivações para o
desenvolvimento de aplicações baseadas
em internet ?
R: Promover a criação de aplicações
com grande disponibilidade, deixando-as
adaptáveis e pertáveis, com fácil
integração entre dispositovos.

2. Quais as funções de um servidor de aplicação?
R: É um ambiante disponibilizado por
um servidor com o intuito de realizar
instalações e execuções de aplicações.
Desta forma centralizando e dividindo
as intalações nos clientes.

3. Quais as diferenças de um página
dinamica e uma página estatica?
R: Uma página estática tem seu conteúdo
inalterado durante o uso do usuário,
enquanto a dinâmica já pode atualizar
ou atualizar suas informações apartir
de uma API por exmeplo.

4. O que é HTML  e como ele é composto ?
R: HTML (Hyper Text Markup Language) é uma
linguagem de marcação de criação de páginas
WEB, composta por Tags para a construção de
componentes que compõe o corpo do site.
